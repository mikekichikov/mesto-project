# Проект: Место

### **Описание:**

Проект интерактивной адаптивной веб-страницы с возможностью выкладывать фотографии и ставить лайки.

### **Проект выполнен с применением:**

- БЭМ
- Медиа-запросы
- Flexbox
- Grid-layouts
- JavaScript
- Webpack
